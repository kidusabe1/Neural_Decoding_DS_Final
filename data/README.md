# Data Directory

This directory contains neural decoding datasets.

## Structure

- `raw/` - Raw data files (pickle, h5)
- `processed/` - Preprocessed data ready for modeling

## Downloading Data

Datasets can be downloaded from:
https://www.dropbox.com/sh/n4924ipcfjqc0t6/AACPWjxDKPEzQiXKUUFriFkJa?dl=0

Place downloaded files in the `raw/` directory.
